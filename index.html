M-CIS-S6-FX3CON

M-CIS-S6-FX3CON is an open source CAMERA development boards for image sensor and high speed interfaces.

   Github page : mgsgo/M-CIS-S6-FX3CON
       Manual, schematics, examples.

   Where to buy : DEVICEMART, product ID : 11678572

   Specification
     Power input : DC 5V 1A maximum, USB mini-B port
     FPGA : XC6SLX16-2FTG256C, Xilinx Spartan6
     Two CIS(CMOS image sensor)s : MT9M114EBLSTCZ-CR1, ONSEMI(Aptina) 1280x960 CIS
     Two M12 lens holders
     Two M12 lenses : Focal length 3.6mm or similar, No IR cut filter
     SPI ROM for FPGA boot : W25Q64FVSSIG (Winbond)
     LPDDR DRAM : MT46H32M16LFBF-5 IT:C, Micron LPDDR
     Gigabit ethernet PHY : KSZ9021RNI, MICREL Gigabit ethernet PHY
     CYUSB3KIT(Cypress FX3 board) connection ports : Two 2.54mm pitch 40pin connectors, Not included CYUSB3KIT board
     HDMI port : Only TMDS signal output through Spartan6 use TMDS_33 IO
         No support HDMI HOT plug detect voltage output
         No HDMI CEC/I2C(DDC) controls

   Examples
     LED_BLINK : LED blink example
         LED blink is the simplest example for M-CIS-S6-FX3CON, toggle LED D8/D9
     UVC_CIS_BYPASS_LEFT_1280x720 : USB3.0 UVC single CIS (AN75779, 1280x720@30fps) example
         Example of UVC(Universal Video Class) camera using 1 image sensor(MT9M114)(CYUSB3KIT-003 required)
         Cypress application note AN75779
     UVC_CIS_LEFT_RIGHT_2560x720 : USB3.0 UVC dual CIS (AN75779, 2560x720@30fps) example
         Example of UVC(Universal Video Class) camera using 2 image sensors(MT9M114)(CYUSB3KIT-003 required)
         Cypress application note AN75779
         Darknet yolo Real time object dection)object detection example compliant(UVC camera, webcam)

     CONFIG_FPGA_OVER_FX3 : Configuring an FPGA over FX3(use AN84868) example(CYUSB3KIT-003 required)
         Example of FX3 configure Spartan6 FPGA(slave serial mode) using AN84868(Configuring FPGA example)
         Cypress application note AN84868
     HDMI_TMDS_VIDEO_OUT : HDMI(TMDS) video output example
         Example of TMDS video signal output through HDMI connector using TMDS_33 IO of Spartan6 FPGA core
     GIGABIT_UDP_THROUGPUT_TEST_BINARY : Gigabit ethernet(1000M) UDP throughput test example
         Example of UDP throughput test using gigabit ethernet MAC/UDP stack (bit file only) (bit file provide only)
         You can use COMBLOCK IP COM-5401SOFT and COM-5402SOFT
     LPDDR_MIG_TRAFFIC : LPDDR test use Xilinx MIG example
         Example of LPDDR write/read example at 200MHz use Xilinx MIG UG388
     SHA1_AUTHENTICATION : SHA-1 EEPROM control example
         Example of SHA-1 EEPROM control (AVNET reference design required)
         S6LX16 PicoBlaze SHA-1 Authentication Design
         XAPP780(for DS2432)

     PMOD compliant module(J11 12pin connector use)
         TOF 1pixel sensor VL53L0X(Not prepared yet)
         Infrared Array Sensor Grid-EYE 8x8pixel sensor AMG8833(Not prepared yet)
         LWIR(Long Wave Infrared) Thermal 80x60pixel sensor FLIR LEPTON(Not prepared yet)
     USB3.0 UVC(Universal Video Class) other image sensor(Not prepared yet)

M-AR0230-DUAL-CAM-UVC

M-AR0230-DUAL-CAM-UVC is a 3840x1080/30fps or 2560x720/60fps camera board with two full HD image sensors.
It can be used directly for image storage or real-time Object Detection using Darknet yolo.

   Specification
     Power input : DC 5V maximum 1A, USB3.0 B port
     FPGA : XC6SLX16-2FTG256C, Xilinx Spartan6
     Two CIS(CMOS image sensor) : AR0230CS, ONSEMI(Aptina) 1920x1080 CIS
     Two M12(diameter 12mm) lens holder
     Two M12 lens : focal length 2.4mm/10.4mm or similar, with IR cut filter
     SPI ROM : W25Q64FVSSIG (Winbond) for FPGA booting

M-AR0230-SINGLE-CAM-HDMI

M-AR0230-SINGLE-CAM-HDMI is a 1920x1080/30~60fps camera board use single full HD image sensor.
Direct connection to a full HD monitor results in a low distortion monitoring system.
