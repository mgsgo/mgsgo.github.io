M-CIS-S6-FX3CON

M-CIS-S6-FX3CON는 이미지센서와 고속 인터페이스를 위한 오픈소스 카메라 개발보드입니다.

   Github page : mgsgo/M-CIS-S6-FX3CON
       매뉴얼, 회로도, 예제.

   구매처 : DEVICEMART, product ID : 11678572

   Specification
     전원 입력 : DC 5V 최대 1A, USB mini-B port
     FPGA : XC6SLX16-2FTG256C, Xilinx Spartan6
     두개의 CIS(CMOS image sensor) : MT9M114EBLSTCZ-CR1, ONSEMI(Aptina) 1280x960 CIS
     두개의 M12(지름 12mm) 렌즈 홀더
     두개의 M12 렌즈 : 초점거리 3.6mm 혹은 유사한 초점거리, IR 차단 필터 없음
     FPGA 부팅을 위한 SPI ROM : W25Q64FVSSIG (Winbond)
     LPDDR DRAM : MT46H32M16LFBF-5 IT:C, Micron LPDDR
     Gigabit ethernet PHY : KSZ9021RNI, MICREL Gigabit ethernet PHY
     CYUSB3KIT-003(Cypress FX3 board) 연결 단자 : 두개의 2.54mm 간격 40핀 커넥터, CYUSB3KIT-003 보드 미포함
     HDMI 단자 : Spartan6에서 출력된 TMDS 신호(TMDS_33 IO) 연결
         HDMI HOT plug detect voltage output 미지원
         HDMI CEC/I2C(DDC) 제어 미지원

   Examples
     LED_BLINK : LED blink 예제
         LED D8, D9가 번갈아가면서 켜지는 예제
     UVC_CIS_BYPASS_LEFT_1280x720 : USB3.0 UVC single CIS (AN75779, 1280x720@30fps) 예제
         UVC(Universal Video Class) 카메라 예제, 이미지센서 1개 사용(MT9M114)(CYUSB3KIT-003 필요함)
         Cypress application note AN75779
     UVC_CIS_LEFT_RIGHT_2560x720 : USB3.0 UVC single CIS (AN75779, 2560x720@30fps) 예제
         UVC(Universal Video Class) 카메라 예제, 이미지센서 2개 사용(MT9M114)(CYUSB3KIT-003 필요함)
         Cypress application note AN75779
         Darknet yolo Real time object dection을 활용한 실시간 object detection 예제 구동 가능

     CONFIG_FPGA_OVER_FX3 : Configuring an FPGA over FX3(use AN84868) 예제 (CYUSB3KIT-003 required)
         AN84868(Configuring FPGA example)에서 FX3가 Spartan6 FPGA를 mode slave serial로 configuration 하는 예제
         Cypress application note AN84868
     HDMI_TMDS_VIDEO_OUT : HDMI(TMDS) video 출력 예제
         Spartan6 FPGA에서 TMDS_33 IO를 이용하여 TMDS 신호를 출력하는 예제 core
     GIGABIT_UDP_THROUGPUT_TEST_BINARY : Gigabit ethernet(1000M) UDP throughput test 예제
         Gigabit ethernet MAC/UDP stack을 이용한 UDP 출력 테스트 (bit 파일 제공) (bit file provide only)
         You can use COMBLOCK IP COM-5401SOFT and COM-5402SOFT
     LPDDR_MIG_TRAFFIC : Xilinx MIG를 이용한 LPDDR test 예제
         Xilinx MIG(UG388)를 이용한 200MHz 에서 작동하는 LPDDR write/read 예제 UG388
     SHA1_AUTHENTICATION : SHA-1 EEPROM 제어 예제
         SHA-1 EEPROM 제어 예제 (AVNET reference design 필요함)
         S6LX16 PicoBlaze SHA-1 Authentication Design
         XAPP780(for DS2432)

     PMOD compliant module(J11 12pin connector use)
         TOF 1pixel sensor VL53L0X(Not prepared yet)
         Infrared Array Sensor Grid-EYE 8x8pixel sensor AMG8833(Not prepared yet)
         LWIR(Long Wave Infrared) Thermal 80x60pixel sensor FLIR LEPTON(Not prepared yet)
     USB3.0 UVC(Universal Video Class) other image sensor(Not prepared yet)
